# CSCI-223-Semester-Project-Algorithm-Design
To run the program, just go ahead and run the Python file. An input  "Search:" will show up on the screen, and go ahead and start to type a word, and there will be suggestions of common words starting with the same letters that you put in. 
The required files are project_final.py (I have changed the library to  the most common English words), and make sure you have nltk downloaded.
I used a Trie combined with a Depth-First Search.
Overall complexity: O(P + R × L)
